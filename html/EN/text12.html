Instalation of necessary development environment based on SCIA OpenAPI (Windows 11, Visual Studio 2022, SCIA Engineer 24.0)

  • Program entry point is in Program.cs file in Main procedure. Placing a breakpoint on line #23 (click on the line and press F9) will cause program to stop at that line
  • Pressing F5 or F11 will step trough the code. F5 will run it up to the next break point, F11 to the next line.
